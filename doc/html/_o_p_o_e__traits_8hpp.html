Detailed Description

Defines core type traits, enumerations, and data structures for the OPOE library.

This header provides essential type definitions and enumerations used throughout the OPOE library, including matrix and vector types based on Eigen, polynomial field types, and various enums for evaluation, integration, and quantization methods.

Dependencies:

  • Eigen library for matrix and vector operations.
  • Standard library features (e.g., std::complex, std::vector, std::array).
  • C++20 concepts for type constraints.
