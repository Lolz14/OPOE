Detailed Description

template<std::size_t N_points, typename T>
class stats::Quantizer< N_points, 1, QuantizationProcedure::Newton, T >

Specialization of Quantizer for 1D using Newton's method.

This class implements a 1D quantizer using Newton's method to minimize the distortion function for a Gaussian distribution. It inherits from QuantizerBase and provides an optimization routine based on analytical gradients and Hessians.

Template Parameters
TScalar type (e.g., float or double).
