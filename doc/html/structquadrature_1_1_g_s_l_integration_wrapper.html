Detailed Description

template<typename R = traits::DataType::PolynomialField>
struct quadrature::GSLIntegrationWrapper< R >

GSLQuadrature class for numerical integration using GSL's adaptive quadrature.

This class provides an adapter for GSL's adaptive quadrature routines, allowing for numerical integration over finite and infinite bounds with specified absolute and relative error tolerances.

Template Parameters
RThe floating-point type used for calculations (default: traits::DataType::PolynomialField).
