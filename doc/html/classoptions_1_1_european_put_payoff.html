Detailed Description

template<typename T = traits::DataType::PolynomialField>
class options::EuropeanPutPayoff< T >

Specialization of IPayoff for European Put options.

This class implements the payoff evaluation for European put options, defined as max(K - S_T, 0), where S_T is the final price of the underlying asset and K is the strike price.

Template Parameters
TThe numeric type used for the strike price and evaluations (default: traits::DataType::PolynomialField).
