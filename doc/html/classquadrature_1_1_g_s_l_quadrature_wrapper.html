Detailed Description

template<typename R = traits::DataType::PolynomialField>
class quadrature::GSLQuadratureWrapper< R >

Wrapper for the GSL adaptive quadrature rule.

Template Parameters
RThe floating-point type used for integration (e.g., double).

This class wraps a GSLQuadrature instance and exposes it through the IQuadratureRule interface. It supports construction from an existing rule or by specifying the desired absolute and relative errors.

  • integrate: Performs numerical integration using the wrapped GSL rule.
  • clone: Creates a deep copy of the wrapper and its underlying rule.
