Detailed Description

template<typename R = traits::DataType::PolynomialField>
class options::CFOptionPricer< R >

CFOptionPricer class for pricing options using the Black-Scholes model.

This class implements the closed-form solution for European options under the Black-Scholes model. It supports both call and put options and provides methods for computing option price and Greeks.

Template Parameters
RThe floating-point type used for calculations (default: traits::DataType::PolynomialField).
