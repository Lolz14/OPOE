Detailed Description

template<typename R = traits::DataType::PolynomialField>
class quadrature::BoostTanhSinhQuadrature< R >

BoostTanhSinhQuadrature class for numerical integration using Boost's tanh_sinh quadrature.

This class provides an adapter for Boost's tanh_sinh quadrature method, allowing for numerical integration over finite and infinite bounds with specified error tolerances.

Template Parameters
RThe floating-point type used for calculations (default: traits::DataType::PolynomialField).

Features:

  • Handles both finite and infinite bounds.
  • Provides a method to integrate a function with specified absolute and relative error tolerances.
  • Uses Boost's tanh_sinh quadrature internally for efficient integration.
