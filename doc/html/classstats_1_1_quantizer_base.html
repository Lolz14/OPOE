Detailed Description

template<std::size_t N_points, std::size_t Dim_val, typename T = traits::DataType::PolynomialField>
class stats::QuantizerBase< N_points, Dim_val, T >

Base class template for quantizer implementations.

This class provides a generic interface and common storage for quantization algorithms. It defines and manages the quantizer grid (centroids) and hints (initial positions).

Template Parameters
TScalar type (e.g., float or double).
N_pointsNumber of quantization points (must be > 0).
Dim_valDimensionality of the input space (must be > 0).
