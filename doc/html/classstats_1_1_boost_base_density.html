Detailed Description

template<typename BoostDist, typename R, typename... ConstructorArgs>
class stats::BoostBaseDensity< BoostDist, R, ConstructorArgs >

Generic wrapper for Boost continuous probability distributions.

Provides a uniform interface for PDF, CDF, quantile, and parameter access. Uses SFINAE to enable distribution-specific properties (e.g., mean, stddev).

Template Parameters
BoostDistThe specific Boost distribution type (e.g., boost::math::normal_distribution).
RNumeric type (defaults to double).
ConstructorArgsConstructor argument types used to instantiate BoostDist.
