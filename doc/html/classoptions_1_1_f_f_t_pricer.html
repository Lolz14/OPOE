Detailed Description

template<typename R = traits::DataType::PolynomialField>
class options::FFTPricer< R >

FFTPricer class for pricing options using Fast Fourier Transform (FFT).

This class implements the FFT method for option pricing, leveraging the characteristic function of the underlying asset. It supports flexible polynomial bases and integrates the expected payoff using configurable quadrature rules.

Template Parameters
RThe floating-point type used for calculations (default: traits::DataType::PolynomialField).
