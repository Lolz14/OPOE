Detailed Description

template<typename R, typename... FuncTypes>
class quadrature::Projector< R, FuncTypes >

Computes the weighted integral of the product of multiple functions.

Accepts a variable number of input functions via its constructor. Calculates Integral( lower, upper, weight_fn(x) * f1(x) * f2(x) * ... * fn(x) dx ) using a provided quadrature rule.

Template Parameters
RThe numeric type for calculations (e.g., double, float).
FuncTypesThe types of the input functions (f1, f2, ... fn). All must be callable with signature R(R).
