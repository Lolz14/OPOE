Detailed Description

template<typename R>
struct polynomials::RecurrenceCoefficients< R >

Structure for defining recurrence coefficients of orthogonal polynomials.

The recurrence takes the form: P_{k+1}(x) = (x - alpha_k(x)) * P_k(x) - beta_k(x) * P_{k-1}(x)
