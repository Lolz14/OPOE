Detailed Description

template<typename R>
class options::MCOptionPricer< R >

Monte Carlo Option Pricer class for pricing options using Monte Carlo simulation.

This class implements the Monte Carlo method for option pricing, simulating paths of the underlying asset and calculating the expected payoff at maturity.

Template Parameters
RThe floating-point type used for calculations (default: traits::DataType::PolynomialField).
