Detailed Description

template<typename T = traits::DataType::PolynomialField>
class SDE::GeometricBrownianMotionSDE< T >

Geometric Brownian Motion SDE model.

Template Parameters
TNumeric type for computations (default: traits::DataType::PolynomialField).

This model represents the classic Geometric Brownian Motion, commonly used for modeling stock prices. It includes methods for drift, diffusion, and characteristic function calculations.
