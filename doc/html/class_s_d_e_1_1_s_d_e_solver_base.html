Detailed Description

template<typename DerivedSolver, SDEModel SdeType, typename R = traits::DataType::PolynomialField>
class SDE::SDESolverBase< DerivedSolver, SdeType, R >

Base class for SDE solvers using CRTP for static polymorphism.

Template Parameters
DerivedSolverThe derived solver class (CRTP pattern).
SdeTypeThe SDE model type satisfying SDEModel concept.
RScalar type for computations (default: traits::DataType::PolynomialField).

Generates Wiener process increments for simulation.

Parameters
dtTime step size.
dW_outMatrix to store increments (WIENER_DIM x num_steps).
num_stepsNumber of time steps.
num_pathsNumber of paths to simulate.
Exceptions
std::invalid_argumentif dt is negative.

This method generates independent Wiener increments for each path and stores them in dW_out. If dt is zero, it initializes dW_out to zero.

Parameters
t_startStarting time.
t_endEnding time.
num_stepsNumber of time steps.
num_pathsNumber of paths to simulate.
observerCallback function with signature void(double time, const SDEVector& state).
dW_optOptional pre-generated Wiener increments (if provided, must match num_paths and num_steps).
Exceptions
std::invalid_argumentif initial state size does not match SDE model dimensions.

This method simulates the SDE over the specified time range and invokes the observer callback at each step with the current time and state vector.
