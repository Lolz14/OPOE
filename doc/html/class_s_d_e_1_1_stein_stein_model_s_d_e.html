Detailed Description

template<typename T = traits::DataType::PolynomialField>
class SDE::SteinSteinModelSDE< T >

Stein-Stein Model SDE implementation.

Template Parameters
TNumeric type for computations (default: traits::DataType::PolynomialField). This model is a specific case of the GenericSVModelSDE with p=1 and q=0.
