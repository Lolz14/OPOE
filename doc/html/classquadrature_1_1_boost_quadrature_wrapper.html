Detailed Description

template<typename R = traits::DataType::PolynomialField>
class quadrature::BoostQuadratureWrapper< R >

Wrapper for the Boost Tanh-Sinh quadrature rule.

Template Parameters
RThe floating-point type used for integration (e.g., double).

This class wraps a BoostTanhSinhQuadrature instance and exposes it through the IQuadratureRule interface. It supports construction from an existing rule or by specifying the desired relative error.

  • integrate: Performs numerical integration using the wrapped Boost rule.
  • clone: Creates a deep copy of the wrapper and its underlying rule.
