\doxysubsection{Detailed Description}
Defines the CFOption\+Pricer class for closed-\/form (Black-\/\+Scholes) option pricing. 

This header provides the implementation of the CFOption\+Pricer class, which computes the price and Greeks of European options using the Black-\/\+Scholes model. The class supports both call and put payoffs, and requires the underlying \mbox{\hyperlink{namespace_s_d_e}{SDE}} model to be a Geometric Brownian Motion (GBM).

Key Features\+:
\begin{DoxyItemize}
\item Pricing of European call and put options using the Black-\/\+Scholes closed-\/form solution.
\item Calculation of option Greeks\+: delta, gamma, vega, theta, and rho.
\item Type-\/safe construction, ensuring only GBM models are accepted.
\item Template parameter for floating-\/point precision.
\end{DoxyItemize}
