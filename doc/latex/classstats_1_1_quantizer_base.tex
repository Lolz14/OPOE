class stats\+::\+Quantizer\+Base$<$ N\+\_\+points, Dim\+\_\+val, T $>$}

Base class template for quantizer implementations. 

This class provides a generic interface and common storage for quantization algorithms. It defines and manages the quantizer grid (centroids) and hints (initial positions).
