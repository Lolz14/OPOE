\doxysubsection{Detailed Description}
Implements quantization algorithms for vector quantization, including Newton\textquotesingle{}s method (1D) and Competitive Learning Vector Quantization (CLVQ) for arbitrary dimensions. 

This header defines the {\ttfamily \mbox{\hyperlink{classstats_1_1_quantizer}{stats\+::\+Quantizer}}} class template and its specializations for different quantization procedures. It provides a flexible framework for optimal quantizer design, supporting both batch (Newton\textquotesingle{}s method for 1D) and online (CLVQ) learning.
