\doxysubsection{Detailed Description}
Provides generic quadrature (numerical integration) adapters for Boost and GSL libraries. 

This header defines two main quadrature classes\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classquadrature_1_1_boost_tanh_sinh_quadrature}{quadrature\+::\+Boost\+Tanh\+Sinh\+Quadrature}}\+: Adapter for Boost.\+Math\textquotesingle{}s tanh\+\_\+sinh quadrature, supporting finite and infinite bounds.
\item \mbox{\hyperlink{classquadrature_1_1_g_s_l_quadrature}{quadrature\+::\+GSLQuadrature}}\+: Adapter for GSL\textquotesingle{}s adaptive quadrature routines, supporting finite and infinite bounds with workspace management.
\end{DoxyItemize}

Dependencies\+:
\begin{DoxyItemize}
\item Boost.\+Math for tanh\+\_\+sinh quadrature.
\item GSL (GNU Scientific Library) for adaptive quadrature.
\item \mbox{\hyperlink{_o_p_o_e__traits_8hpp}{traits/\+OPOE\+\_\+traits.\+hpp}} for type definitions.
\end{DoxyItemize}

Features\+:
\begin{DoxyItemize}
\item Type-\/generic (templated on floating-\/point type R).
\item Handles both finite and infinite integration limits.
\item Error control via absolute and relative tolerances.
\item Exception-\/safe resource management for GSL workspaces.
\item C++11/14 compatible, uses std\+::function for integrands.
\end{DoxyItemize}
