\doxysubsection{Detailed Description}
Provides constexpr functions to compute monomials of a given degree. 

This header defines a set of template functions in the {\ttfamily polynomials} namespace for evaluating monomials (i.\+e., powers of a variable) at compile time. The functions use template metaprogramming and type traits to recursively compute x$^\wedge$N for a given value x and non-\/negative integer N.
