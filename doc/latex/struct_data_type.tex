\doxysubsection{Detailed Description}
Central container of type aliases for commonly used matrix/vector structures in OPOE. 

These types are based on Eigen and are designed for dynamic sizing and numerical efficiency, targeting polynomial manipulation and high-\/dimensional vector/matrix operations.
