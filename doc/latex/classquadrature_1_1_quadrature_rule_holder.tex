\subsubsection*{template$<$typename R$>$\newline
class quadrature\+::\+Quadrature\+Rule\+Holder$<$ R $>$}

A holder class for different numerical quadrature rules. 

This class encapsulates various quadrature rules through runtime polymorphism. It enables selection and usage of a rule at runtime, while maintaining a consistent interface.
