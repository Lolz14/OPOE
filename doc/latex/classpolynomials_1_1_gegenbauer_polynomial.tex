class polynomials\+::\+Gegenbauer\+Polynomial$<$ N, R $>$}

Gegenbauer polynomials with parameter λ (lambda). 

Gegenbauer polynomials are a generalization of Legendre polynomials and are orthogonal on the interval \mbox{[}-\/1, 1\mbox{]} with respect to the weight function (1 -\/ x$^\wedge$2)$^\wedge$(λ -\/ 0.\+5).
