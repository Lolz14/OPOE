\mbox{\hyperlink{classquadrature_1_1_boost_tanh_sinh_quadrature}{Boost\+Tanh\+Sinh\+Quadrature}} class for numerical integration using Boost\textquotesingle{}s tanh\+\_\+sinh quadrature. 

This class provides an adapter for Boost\textquotesingle{}s tanh\+\_\+sinh quadrature method, allowing for numerical integration over finite and infinite bounds with specified error tolerances.
