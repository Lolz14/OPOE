Interface for option payoff functions.

This is the abstract base class for all option payoff types. Exposed to Python as {\ttfamily IPayoff}.

Payoffs define how the terminal asset price {\ttfamily S\+\_\+T} is transformed into a cashflow (e.\+g. max(S\+\_\+T -\/ K, 0) for a call).

European Call option payoff.

Payoff function\+: $\max(S_T - K, 0)$

European Put option payoff.

Payoff function\+: $\max(K - S_T, 0)$
