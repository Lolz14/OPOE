\doxysubsection{Detailed Description}
Stochastic Differential Equation (\mbox{\hyperlink{namespace_s_d_e}{SDE}}) solver framework with support for multiple numerical schemes. 

This header defines a flexible and extensible framework for simulating SDEs using various numerical methods, including Euler-\/\+Maruyama, Milstein, and Interpolated Kahl-\/\+Jackel schemes. The framework leverages C++20 concepts for compile-\/time constraints, Eigen for linear algebra, and Open\+MP for parallelism.

Key Features\+:
\begin{DoxyItemize}
\item Thread-\/local random number generation utilities for reproducible and parallel-\/safe simulations.
\item Strongly-\/typed \mbox{\hyperlink{namespace_s_d_e}{SDE}} model concepts (SDEModel, SVSDEModel, Milstein1\+DSDEModel, Milstein2\+DSVSDEModel) for compile-\/time validation.
\item CRTP-\/based base solver class (SDESolver\+Base) for static polymorphism and code reuse.
\item Concrete solver implementations for Euler-\/\+Maruyama, Milstein, and Interpolated Kahl-\/\+Jackel methods.
\item Support for multi-\/path, multi-\/step simulations with observer callbacks and optional pre-\/generated Wiener increments.
\item Full truncation to ensure non-\/negativity of state variables where appropriate.
\end{DoxyItemize}

Template Parameters\+:
\begin{DoxyItemize}
\item Sde\+Type\+: The \mbox{\hyperlink{namespace_s_d_e}{SDE}} model type, which must satisfy the relevant SDEModel concept.
\item R\+: Scalar type for computations (default\+: traits\+::\+Data\+Type\+::\+Polynomial\+Field).
\end{DoxyItemize}
