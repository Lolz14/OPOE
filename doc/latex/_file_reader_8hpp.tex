\doxysubsection{Detailed Description}
Provides functionality to read quantization grid data from a file into Eigen-\/based structures. 

This header defines the \mbox{\hyperlink{struct_quantization_grid}{Quantization\+Grid}} struct and the read\+Quantization\+Grid function template, which are used to load quantization grid data (including coordinates, weights, and distortions) from a file into Eigen matrices and vectors for efficient numerical processing.
