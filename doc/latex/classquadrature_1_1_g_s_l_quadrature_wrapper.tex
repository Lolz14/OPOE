class quadrature\+::\+GSLQuadrature\+Wrapper$<$ R $>$}

Wrapper for the GSL adaptive quadrature rule. 


\begin{DoxyTemplParams}{Template Parameters}
{\em R} & The floating-\/point type used for integration (e.\+g., double).\\
\hline
\end{DoxyTemplParams}
This class wraps a {\ttfamily \mbox{\hyperlink{classquadrature_1_1_g_s_l_quadrature}{GSLQuadrature}}} instance and exposes it through the {\ttfamily \mbox{\hyperlink{classquadrature_1_1_i_quadrature_rule}{IQuadrature\+Rule}}} interface. It supports construction from an existing rule or by specifying the desired absolute and relative errors.


\begin{DoxyItemize}
\item {\ttfamily integrate}\+: Performs numerical integration using the wrapped GSL rule.
\item {\ttfamily clone}\+: Creates a deep copy of the wrapper and its underlying rule.
