\subsubsection*{template$<$typename R, typename... Func\+Types$>$\newline
class quadrature\+::\+Projector$<$ R, Func\+Types $>$}

Computes the weighted integral of the product of multiple functions. 

Accepts a variable number of input functions via its constructor. Calculates Integral( lower, upper, weight\+\_\+fn(x) $\ast$ f1(x) $\ast$ f2(x) $\ast$ ... $\ast$ fn(x) dx ) using a provided quadrature rule.


\begin{DoxyTemplParams}{Template Parameters}
{\em R} & The numeric type for calculations (e.\+g., double, float). \\
\hline
{\em Func\+Types} & The types of the input functions (f1, f2, ... fn). All must be callable with signature R(\+R). \\
\hline
