class options\+::\+European\+Put\+Payoff$<$ T $>$}

Specialization of \mbox{\hyperlink{classoptions_1_1_i_payoff}{IPayoff}} for European Put options. 

This class implements the payoff evaluation for European put options, defined as max(K -\/ S\+\_\+T, 0), where S\+\_\+T is the final price of the underlying asset and K is the strike price.
