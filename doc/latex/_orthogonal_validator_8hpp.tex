\doxysubsection{Detailed Description}
Provides utilities for validating domains and parameters of orthogonal polynomials. 

This header defines concepts, structures, and classes to facilitate the validation of parameters and domains for various families of orthogonal polynomials (e.\+g., Jacobi, Laguerre, Hermite, Gegenbauer).

It includes\+:
\begin{DoxyItemize}
\item Function type alias for recurrence coefficients.
\item Domain\+Interval for interval validation (with floating-\/point support).
\item Exception types for domain violations.
\item Parameter base structure and specific parameter types for common polynomial families.
\item Polynomial\+Domain\+Validator class template for parameter validation and access. 
\end{DoxyItemize}
