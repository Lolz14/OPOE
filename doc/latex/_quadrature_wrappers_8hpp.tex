\doxysubsection{Detailed Description}
Provides wrapper classes for different quadrature (numerical integration) backends. 

This header defines wrapper classes for Boost and GSL quadrature rules, allowing them to be used through a common interface. The wrappers manage the underlying rule objects and provide methods for integration and cloning.
