\doxysubsection{Detailed Description}
Stochastic Differential Equation (\mbox{\hyperlink{namespace_s_d_e}{SDE}}) models for financial applications. 

This header defines a set of classes and interfaces for modeling various stochastic processes commonly used in mathematical finance, such as Geometric Brownian Motion, Heston, Stein-\/\+Stein, Hull-\/\+White, and Jacobi models. These models are implemented as SDEs and provide interfaces for drift, diffusion, and their derivatives, as well as characteristic functions and generator matrices.

Main Components\+:
\begin{DoxyItemize}
\item ISDEModel\+: Abstract base class for \mbox{\hyperlink{namespace_s_d_e}{SDE}} models, defining the interface for drift, diffusion, their derivatives, characteristic functions, and initial state management.
\item Geometric\+Brownian\+Motion\+SDE\+: Implements the classic GBM model for asset prices.
\item Generic\+SVModel\+SDE\+: Generic stochastic volatility model supporting flexible exponents for volatility.
\item Heston\+Model\+SDE\+: Specialization of Generic\+SVModel\+SDE for the Heston model (square-\/root volatility).
\item Stein\+Stein\+Model\+SDE\+: Specialization for the Stein-\/\+Stein model (OU volatility).
\item Hull\+White\+Model\+SDE\+: Specialization for the Hull-\/\+White model (linear volatility).
\item Jacobi\+Model\+SDE\+: Specialization for the Jacobi model, with bounded variance process.
\end{DoxyItemize}

Features\+:
\begin{DoxyItemize}
\item Support for both real and complex-\/valued state vectors and matrices.
\item Calculation of drift, diffusion, and their derivatives for use in numerical schemes.
\item Characteristic function computation for Fourier-\/based pricing methods.
\item Construction of generator matrices for polynomial expansion methods.
\item Parameter validation and error handling for model consistency.
\end{DoxyItemize}

Template Parameter\+:
\begin{DoxyItemize}
\item T\+: Numeric type (default\+: traits\+::\+Data\+Type\+::\+Polynomial\+Field), allowing for flexibility in precision.
\end{DoxyItemize}

Usage\+:
\begin{DoxyItemize}
\item Instantiate a model class with appropriate parameters and initial state.
\item Use the drift and diffusion methods for simulation or numerical solution of SDEs.
\item Use the characteristic function for pricing or calibration tasks.
\item Use generator\+\_\+G for polynomial expansion or spectral methods.
\end{DoxyItemize}
