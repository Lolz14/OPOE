\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{classpolynomials_1_1_polynomial}{polynomials\+::\+Polynomial}} template class and related polynomial operations. 

This header provides a generic, fixed-\/degree polynomial class template with support for various arithmetic operations, evaluation methods, and utilities. The implementation leverages Eigen/\+Armadillo for efficient array operations and supports both Horner\textquotesingle{}s and direct evaluation methods. Additional features include conversion to std\+::function, pretty-\/printing, and compile-\/time degree information.

Dependencies\+:
\begin{DoxyItemize}
\item Armadillo/\+Eigen for array operations.
\item C++17 or later (C++20 for comparison operators).
\item Custom traits and utility headers.
\end{DoxyItemize}

Main features\+:
\begin{DoxyItemize}
\item Template class {\ttfamily Polynomial$<$N, R$>$} for polynomials of degree N over field R.
\item Supports construction from coefficient arrays, copy/move semantics, and assignment from polynomials of lower degree.
\item Evaluation via operator() with selectable method (Horner/direct).
\item Conversion to std\+::function for functional programming use cases.
\item Arithmetic operations\+: addition, subtraction, multiplication (including FFT-\/based), and scalar operations.
\item Derivative computation via the {\ttfamily der$<$M$>$(p)} template function.
\item Pretty-\/printing via operator$<$$<$.
\item C++20 three-\/way comparison operator (if available).
\end{DoxyItemize}
