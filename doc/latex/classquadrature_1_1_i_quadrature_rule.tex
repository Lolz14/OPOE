class quadrature\+::\+IQuadrature\+Rule$<$ Numeric\+Type $>$}

Abstract interface for numerical quadrature rules. 

This interface defines the contract for numerical integration rules, allowing for flexible and extensible integration methods. It supports both finite and infinite bounds, and provides a method to integrate a function with specified error tolerances.
