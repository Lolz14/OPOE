class options\+::\+CFPricer$<$ R $>$}

\mbox{\hyperlink{classoptions_1_1_c_f_pricer}{CFPricer}} class for pricing options using the Black-\/\+Scholes model. 

This class implements the closed-\/form solution for European options under the Black-\/\+Scholes model. It supports both call and put options and provides methods for computing option price and Greeks.
