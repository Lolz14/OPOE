class quadrature\+::\+Boost\+Quadrature\+Wrapper$<$ R $>$}

Wrapper for the Boost Tanh-\/\+Sinh quadrature rule. 


\begin{DoxyTemplParams}{Template Parameters}
{\em R} & The floating-\/point type used for integration (e.\+g., double).\\
\hline
\end{DoxyTemplParams}
This class wraps a {\ttfamily \mbox{\hyperlink{classquadrature_1_1_boost_tanh_sinh_quadrature}{Boost\+Tanh\+Sinh\+Quadrature}}} instance and exposes it through the {\ttfamily \mbox{\hyperlink{classquadrature_1_1_i_quadrature_rule}{IQuadrature\+Rule}}} interface. It supports construction from an existing rule or by specifying the desired relative error.

\item {\ttfamily clone}\+: Creates a deep copy of the wrapper and its underlying rule.
