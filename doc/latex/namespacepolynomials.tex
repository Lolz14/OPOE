\doxysubsection{Detailed Description}
Validates and debugs polynomial domain parameters. 

This file contains the implementation of member functions for the \mbox{\hyperlink{classpolynomials_1_1_polynomial_domain_validator}{Polynomial\+Domain\+Validator}} class template, which is responsible for validating the parameters of a polynomial domain and providing debugging information.

Dependencies\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{_orthogonal_validator_8hpp}{Orthogonal\+Validator.\+hpp}}\+: Header file for the \mbox{\hyperlink{classpolynomials_1_1_polynomial_domain_validator}{Polynomial\+Domain\+Validator}} class.
\end{DoxyItemize}


\begin{DoxyTemplParams}{Template Parameters}
{\em R} & The return type or result type associated with the validator. \\
\hline
{\em Params} & Variadic template parameters representing the types of the parameters to validate.\\
\hline
\end{DoxyTemplParams}
Member Functions\+:
\begin{DoxyItemize}
\item validate\+Parameters()\+: Checks if all parameters are valid according to their domain. Throws a std\+::runtime\+\_\+error with a detailed error message if any parameter is invalid.
\item debug\+Parameters()\+: Outputs the name and value of each parameter to std\+::cout for debugging purposes.
\item build\+Error\+Message()\+: Constructs a detailed error message listing all invalid parameters, their values, and the expected domain.
\end{DoxyItemize}

Usage\+: Instantiate \mbox{\hyperlink{classpolynomials_1_1_polynomial_domain_validator}{Polynomial\+Domain\+Validator}} with the appropriate parameter types, then call validate\+Parameters() to ensure all parameters are valid before proceeding.
