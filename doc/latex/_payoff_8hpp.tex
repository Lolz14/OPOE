\doxysubsection{Detailed Description}
Defines interfaces and concrete classes for option payoff evaluation in quantitative finance. 

This header provides a generic interface ({\ttfamily IPayoff}) for evaluating the payoff of financial derivatives, such as options, given the final price or log-\/price of the underlying asset. It also includes concrete implementations for European call and put options, supporting both price and log-\/price evaluation.

Classes\+:
\begin{DoxyItemize}
\item IPayoff$<$\+T$>$\+: Abstract base class for option payoffs, supporting polymorphic evaluation and cloning.
\item European\+Call\+Payoff$<$\+T$>$\+: Implements the payoff for a European call option.
\item European\+Put\+Payoff$<$\+T$>$\+: Implements the payoff for a European put option.
\end{DoxyItemize}

Usage\+:
\begin{DoxyItemize}
\item Instantiate concrete payoff classes with the desired strike price.
\item Use {\ttfamily evaluate} for payoff calculation from price, or {\ttfamily evaluate\+\_\+from\+\_\+log} for log-\/price.
\item Use {\ttfamily clone} for polymorphic copying.
\end{DoxyItemize}
