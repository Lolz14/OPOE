\doxysubsection{Detailed Description}
Defines the Mixture\+Density class template for representing and manipulating mixtures of probability density functions (PDFs) with orthogonal polynomial bases. 

Dependencies\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{_density_base_8hpp}{Density\+Base.\+hpp}}\+: Base class for probability density functions.
\item \mbox{\hyperlink{_orthogonal_polynomials_8hpp}{Orthogonal\+Polynomials.\+hpp}}\+: For polynomial operations and orthogonal polynomial bases.
\end{DoxyItemize}

This header provides a flexible framework for constructing mixtures of various probability distributions (e.\+g., Normal, Gamma, Beta) and associating each component with its corresponding orthogonal polynomial basis (e.\+g., Hermite, Laguerre, Jacobi). The Mixture\+Density class supports\+:
\begin{DoxyItemize}
\item Validation and storage of mixture weights and component densities.
\item Automatic selection and construction of the appropriate polynomial basis for each component via Density\+To\+Poly\+Traits.
\item Computation of recursion coefficients for orthonormal polynomial basis construction using the mixture\textquotesingle{}s weighted inner product.
\item Construction of the orthonormal polynomial basis for the mixture, including storage of polynomial coefficients and function representations.
\item Calculation of the mixture\textquotesingle{}s PDF and CDF at arbitrary points.
\item Access to component parameters, polynomial bases, Jacobi matrices, and projection matrices.
\item Prevention of copy semantics for safety, with move semantics enabled.
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Requires C++17 or later for features such as std\+::variant, std\+::apply, and fold expressions.
\item Relies on Eigen for matrix and vector operations, and Open\+MP for parallelization.
\item Assumes that Density\+Type classes provide methods\+: pdf(x), cdf(x), get\+Mu(), get\+Sigma(), get\+Alpha(), get\+Beta(), get\+Shape(), get\+Scale(), get\+Domain(), get\+Distribution(), and get\+Constructor\+Parameters().
\item The polynomial basis construction is tailored to the type of each component via Density\+To\+Poly\+Traits specializations.
\end{DoxyItemize}
\end{DoxyNote}
